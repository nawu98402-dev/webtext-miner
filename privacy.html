隐私政策 (Privacy Policy)

最近更新日期：2026年2月24日

欢迎使用 WebText Miner。我们深知隐私对您的重要性，本政策旨在透明地说明我们在您使用本工具时如何处理信息。

1. 信息收集

我们的工具采用纯前端解析技术或透明的 API 请求。我们不会收集、存储或上传您输入的任何目标网页 URL，也不会记录您提取出的任何文本内容。

2. Cookie 的使用

本网站本身不使用 Cookie 跟踪用户。但请注意，作为未来可能集成 Google AdSense 的站点，Google 可能会使用 Cookie 根据您访问本网站或其他网站的情况在您的浏览器上投放广告。您可以随时在 Google 广告设置中选择停用个性化广告。

3. 数据安全

由于所有处理过程均在您的本地浏览器完成（或通过加密链接转发），您的数据不会被保存在我们的服务器上，从源头上保障了您的信息安全。

4. 第三方链接

本工具仅用于网页文字提取。通过本工具访问的任何第三方网页均受该网页自身隐私政策的约束，我们建议您在提取内容时尊重原作者的版权及隐私声明。

5. 联系我们

如果您对本隐私政策有任何疑问，请通过 GitHub 仓库的 Issues 页面与我们联系。
